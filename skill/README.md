# react-three-game Skills

Claude Code skills for working with [react-three-game](https://github.com/prnth/react-three-game), a JSON-first 3D game engine built on React Three Fiber, WebGPU, and Rapier Physics.

## Installation

```bash
npx skills add prnth/react-three-game
```

## What's Included

When installed, Claude Code gains knowledge of:

- **GameObject schema** - TypeScript interface for game objects
- **Prefab JSON format** - Structure for scene definitions
- **Built-in components** - Transform, Geometry, Material, Physics, Model, SpotLight
- **Custom component pattern** - How to create Editor/View components with FieldRenderer
- **Field types** - Available field types for editor UI generation
- **Tree utilities** - findNode, updateNode, deleteNode, cloneNode

## Usage

Once installed, Claude Code will automatically apply this skill when working with react-three-game. Example prompts:

- "Create a bouncing ball with physics"
- "Add a custom Rotator component that spins objects"
- "Set up a ground plane with a grid material"
- "Help me structure a scene with multiple nested objects"
